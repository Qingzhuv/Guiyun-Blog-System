# ☁️ GUIYUN 归云：轻量化物理路由博客系统 (V1.0)

本项目是由 **WESTCRAN 西鹤软件** 开发的实验性个人表达与创作平台。不同于传统动态博客，**归云** 采用创新的“物理目录重定向”逻辑，致力于为创作者提供一个更纯粹、更具掌控感的书写环境。

## ✨ 核心特性

- **物理路径隔离**：采用“一文一目录”结构，自动生成 `s/slug/index.php` 物理路径，天生具备良好的 SEO 友好性与结构透明度。
- **Markdown 驱动**：集成 `Marked.js` 高效渲染引擎，支持实时预览，让你的创作回归排版之美。
- **一键初始化**：内置安全安装引导程序，自动化完成环境检测、建库建表及配置文件生成，真正实现“秒级部署”。

## 🛠️ 环境要求

- **服务器**：PHP 8.1+ (推荐使用 PHP 8.1/8.2，必须开启 `mysqli` 扩展)
- **数据库**：MySQL 5.7+
- **部署环境**：支持 Apache / Nginx (需确保 Web 用户对根目录有写入权限以生成物理路径)。

## 🚀 快速部署步骤

### 1. 源码上传

将本项目代码下载并解压至服务器 Web 根目录。

### 2. 权限设定

确保 Web 运行环境对以下目录具有 **读写权限**：

- 项目根目录 (用于生成 `config.php`)
- `s/` 目录 (用于存放自动生成的文章文件夹)

### 3. 一键初始化

在浏览器中访问：`http://您的域名/install.php`。 系统会自动引导您完成数据库对接与管理员账号预设。**部署成功后，系统会锁定安装权限，建议手动删除 `install.php` 以绝后患。**

## 📂 访问说明

- **前台门户首页**：`http://您的域名/`
- **后台管理入口**：`http://您的域名/admin.php`
- **默认文章路径**：`http://您的域名/s/test/`
- **默认管理员**：安装时自定义 (请务必牢记)
